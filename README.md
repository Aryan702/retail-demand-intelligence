# Retail Demand Intelligence Platform

## Overview

Built an end-to-end retail demand forecasting system using Python and machine learning.

The project simulates retail demand across multiple stores and products, engineers time-series features, trains forecasting models, and generates inventory recommendations.

## Problem Statement

Retail businesses often struggle to forecast demand accurately, resulting in stockouts or excess inventory.

This project helps identify demand patterns and supports inventory decision-making.

## Key Features

✅ Multi-store, multi-product retail dataset

✅ Feature engineering:
- Lag features
- Rolling averages
- Growth rate

✅ Machine learning forecasting:
- Random Forest
- XGBoost

✅ Automatic model selection based on MAE

✅ Inventory recommendation engine:
- Increase stock
- Reduce inventory
- Monitor

## Tech Stack
